About

Our eCommerce website is a dynamic and user-friendly platform built using Flask, HTML, Bootstrap, and MySQL. Designed for seamless shopping, it features a responsive interface, secure transactions, and efficient product management. The backend, powered by Flask, ensures smooth functionality, while Bootstrap enhances the design for an intuitive user experience. MySQL handles data storage, managing product listings, user accounts, and order details in real time. With a focus on speed, reliability, and scalability, this project aims to provide a seamless and engaging shopping experience for customers.
